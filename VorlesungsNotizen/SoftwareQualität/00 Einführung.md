# User Stories
## 1 - Scoring
Ich als Kunde möchte das das Scoring eine gute Ersteinschätzung bietet, anhand derer ich besser und schneller entscheiden kann. 

## 2 - Datenqualität
Ich möchte mich auf die Antworten verlassen können, die ich vom System erhalte

## 3 - Transparenz
Als Sachbearbeiter möchte ich die Top-3-Einflussfaktoren für ein Scoring-Ergebnis einsehen, damit ich die Entscheidung gegenüber dem Endkunden fundiert begründen kann.
